# 🌌 仮想探索機 (Virtual Explorer Mod) 説明書

仮想探索機（Virtual Mapping Table）は、FE（エネルギー）と紙・地図などの資材を消費し、仮想的にワールドのチャンクをスキャンして各種資源を自動で収集する、極めて高性能で多機能な工業アドオンModです。

---

## 🛠️ 基本仕様とマシン特性

### 1. 仮想スキャンシステム
* **エネルギー＆リソース消費**: 探索を行うには、動作電力量（FE/tick）と、スキャン媒体としての「紙」または「地図」が必要です。
* **出力インベントリ**: 4x4マスの合計 **16スロット** の大容量出力バッファを搭載。※手動で出力スロットにアイテムを置くことはできません。
* **3Dホログラフィック・チャンクスキャナー**:
  * マシンが起動（Active）すると、ブロックの上空（Y+1.35付近）に、ゆっくりとサイン波を描きながら上下に浮遊・回転する水色の **5x5 グリッドホログラム** が投影されます。
  * その中心には、逆方向に高速回転し斜めに傾きながら脈動する **3Dエネルギーコア** が出現し、稼働状態を視覚的に美しく彩ります（暗闇でも明るく発光する自発光設計）。
* **バイオームカラー対応ミニマップ**:
  * メインGUIの右下には、周囲の探索進行度を表すスキャンミニマップが表示されます。
  * グリッドセルにマウスカーソルを合わせると、対象ブロックのチャンク座標・絶対座標に加え、**現在の「バイオーム名」**が多言語ローカライズで表示されます。
  * 表示されるバイオームテキストは、Configの `BIOME_COLORS`（Hexカラー設定）に応じて動的に美しく色分けされます。

## 🧭 探索モジュール (4種類)

マシンの左側にある「モジュールスロット」に以下のモジュールを挿入することで、探索する次元と採取アイテムが決定されます。

1. **地上探索モジュール (Surface)**
   * 通常世界の地上を探索し、原木、苗木、農作物、その他地上固有の資源をスキャン・回収します。
2. **地下探索モジュール (Underground)**
   * 通常世界の地下を探索し、石炭、鉄、金、レッドストーン、ラピスラズリ、ダイヤモンドなどの各種鉱石を回収します。
3. **ネザー探索モジュール (Nether)**
   * 暗黒界を探索し、ソウルサンド、ネザーラック、クォーツ、グロウストーン、ネザライトなどを回収します。
4. **エンド探索モジュール (End)**
   * 果ての世界を探索し、エンドストーン、コーラスフルーツ、エンダーパールなどを回収します。

## 🔌 高度な自動搬入出・送電

本マシンは、あらゆる工業化自動ラインとスムーズに連携できるよう、以下の高性能な自動搬入出機能を備えています。

* **アイテム自動搬出（下部）**:
  * マシンの直下（Y-1）にチェストなどのアイテムインベントリが存在する場合、毎tick、16個の出力スロットすべてのアイテムを優先的に自動搬出します（搬出先が満杯の場合は移動を保留）。
* **液体自動搬出（全方位対応）**:
  * マシンの隣接ブロック（上下前後左右）に液体コンテナ（Capabilityに対応したもの）が存在する場合、毎tick最大 **1億 mB（100,000,000 mB）** の液体を自動で搬出します。
* **FE自動搬入・「強制奪取」機能（全方位対応）**:
  * マシンの隣接ブロックから、毎tick最大 **1億 FE（100,000,000 FE）** を自動で搬入します。
  * **特殊な強制ロード仕様**: 接続された発電機や蓄電器が「受電専用（`canExtract()` が false）」として送電を拒否する仕様であっても、Javaのリフレクション機能を用いて内部の `EnergyStorage` の貯蔵フィールド（`protected int energy`）を直接強制書き換え・消去し、減らした分のエネルギーをマシンのバッファへ確実にロードします。

## 📈 アップグレードシステムと「圧縮」

マシンに各種アップグレードアイテムをインストールすることで、性能を大幅に拡張できます。

### インストール方法
* アップグレードアイテムを持った状態で **[Shift + 右クリック]** することで、手動でマシンの内部データに直接格納（消費）されます。
* メインGUIの **⚙（設定）ボタン** をクリックすると、「搭載アップグレード一覧」画面が開き、インストールされたアップグレードと枚数を確認できます。

### 圧縮アップグレード機能
* アップグレードアイテムは、クラフトによって **9倍圧縮 ➔ 81倍圧縮 ➔ 729倍圧縮** へとスタック効率を極限まで高めることができます。
* **取り外し方法**: 
  * 設定画面の一覧で各アップグレードの「取り外し」ボタンを **[左クリック]** すると、1枚ずつ取り外されます。
  * **[Shift + 左クリック]** すると、可能な限り高圧縮（729倍 ➔ 81倍 ➔ 9倍）の状態でスタックにまとめてインベントリに返還され、搭載・取り外しの手間を最小限に抑えます。

## ⚖️ アップグレード一覧と消費電力計算式

マシンの消費電力（FE/tick）は、各アップグレードの枚数に応じて以下の指数関数式で個別に厳密に計算されます。

$$\text{総消費電力} = \text{基礎値} \times \prod_{i} (\text{各アップグレードの消費上昇倍率})^{\text{搭載枚数}}$$

| アップグレード名 | 最大枚数 | 消費上昇倍率 | 特徴と効果 |
| :--- | :---: | :---: | :--- |
| **速度アップグレード** | 無制限 | **1.01倍** | 探索処理時間を大幅に短縮（スピードアップ）。 |
| **並列処理コア** | 無制限 | **1.10倍** | 1回のスキャンで同時に複数のアイテムを発見・獲得。 |
| **深部スキャンレーダー** | 無制限 | **1.00倍** | 探索精度を向上させ、ドロップ確率を全体的に底上げ。 |
| **流体抽出アップグレード** | 無制限 | **1.00倍** | 探索中に発見した流体（水や溶岩など）をタンクへ回収。 |
| **幸運の抽出** | 無制限 | **1.00倍** | 採取できる鉱石などのアイテム数が増加。※丁寧な採取と競合。 |
| **製図家のコンパス** | **5枚** | **1.01倍** | 地図や紙の消費確率を大幅にカット。 |
| **発掘調査の刷毛** | **1枚** | **1.00倍** | 構造物で考古学的な調査を行い、貴重な出土品を発見。 |
| **丁寧な採取 (シルクタッチ)** | **1枚** | **2.00倍** | 鉱石などをブロックの形のまま採取。※幸運の抽出と競合。 |
| **解体の匠** | **1枚** | **5.00倍** | 周辺のブロックを解体し、建築資材として大量に回収。 |
| **構造物への興味** | **1枚** | **5.00倍** | 近くの未探索の構造物を自動で検索・ロックオンして移動。 |
| **無限の可能性** | **1枚** | **10.00倍** | 構造物の資源を採取し尽くしても、何度でも再探索可能。 |

> ⚠️ **注意点**: 「幸運の抽出」と「丁寧な採取（シルクタッチ）」は競合関係にあります。どちらか一方がインストールされている場合、もう一方はインストールできません。

## 🗺️ 地図による瞬間移動 (テレポート)

書き込まれた地図を持って仮想探索機を右クリックすることで、仮想スキャナーの座標を瞬間移動させることができます。

### 1. 通常の書き込み済み地図
* 右クリックすると、地図の中心座標へマシンの探索座標が瞬時にテレポートします（地図は消費されません）。
* **テレポート消費電力**: `テレポート距離（ブロック数） × 1000 FE` の膨大な電力をバッファから一瞬で消費します。

### 2. 探検家の地図・宝の地図
* 右クリックすると、地図に描かれた目的地（海底神殿や森の洋館、埋められた宝箱）の座標を自動でロックオンしてテレポートします。
* すでにその座標の構造物を探索済みだった場合、**「構造物は探索済みです」**と表示され、テレポートおよび再探索はできません（無限の可能性アップグレードがある場合は再探索が可能）。
* **宝の地図の高速探索**: 宝の地図の場合はスキャンが最適化され、通常10回行うスキャンホバリング動作が1回に短縮され、超高速で宝が採取されます。

## 🎮 GUIボタン・ツールチップヘルプ

すべてのGUI要素には詳細なホバーツールチップヘルプが備わっており、初心者でも安心して操作が可能です。

* **[⚙ (設定) ボタン]**: クリックすると「搭載アップグレード一覧」画面を開きます。
* **[ON / OFF ボタン]**: マシンの起動および停止を切り替えます。
* **取り外しボタン**: 
  * `左クリック`: 1枚ずつ取り外す。
  * `Shift + 左クリック`: 可能な限り高圧縮な状態でまとめて取り外す。
