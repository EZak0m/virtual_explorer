# バーチャル・エクスプローラー アイテムデザインガイドブック

本Modの世界観（魔導機械・製図・測量）に基づいた、各アイテムのビジュアルデザイン案です。

## 🗺️ 探索モジュール (Explorer Modules)
ベース：精密な書き込みがなされた「地図」

*   **地上探索モジュール (Surface)**
    *   デザイン：明るい緑色の地図。中央に「広葉樹（オーク）」のアイコンと、太陽のシンボルが描かれている。
*   **地下探索モジュール (Underground)**
    *   デザイン：暗灰色の地図。中央に「深層岩」の断面図と、光り輝く「鉄・金の粒」が散りばめられている。
*   **ネザー探索モジュール (Nether)**
    *   デザイン：赤褐色の地図。端が少し焦げており、中央に「ネザーラック」と「マグマの渦」が描かれている。
*   **エンド探索モジュール (End)**
    *   デザイン：薄紫色の地図。中央に「エンドストーン」と、浮遊する「コーラスフルーツ」のシルエットが描かれている。

## ⚙️ アップグレード (Upgrades)
ベース：真鍮や銅で作られた「観測計器・部品」

*   **スピードアップグレード (Speed)**
    *   デザイン：金色の歯車。中心に「青い稲妻」の刻印があり、回転しているようなエフェクト。
*   **製図家のコンパス (Recycle)**
    *   デザイン：真鍮製のコンパス（方位磁石）。針がリサイクルマーク（循環する矢印）の形をしている。
*   **深部スキャンレーダー (Radar)**
    *   デザイン：銅製のアンテナ。中心から緑色の波紋（レーダー波）が出ている液晶パネル。
*   **並列処理コア (Parallel)**
    *   デザイン：アメジストを核にした魔導演算チップ。青白く発光している。
*   **流体サンプラー (Fluid)**
    *   デザイン：ガラス管とバケツが組み合わさった計器。中に青い液体が見える。
*   **発掘調査の刷毛 (Archeology)**
    *   デザイン：考古学用のブラシ。地図の汚れを払うようなポーズ。
*   **解体業者モジュール (Demolition)**
    *   デザイン：TNTとツルハシが交差したマーク。爆発の衝撃波を模したオレンジ色の装飾。
*   **構造物関心アップグレード (Structure Interest)**
    *   デザイン：虫眼鏡（ルーペ）。レンズ越しに「チェスト」が拡大されている。
*   **無限の可能性 (Infinite)**
    *   デザイン：無限（∞）のマーク。虹色に輝くクリスタル。
*   **幸運の観測儀 (Fortune)**
    *   デザイン：クローバーの装飾が施されたアストロラーベ（観測儀）。
*   **精密採取（シルクタッチ）**
    *   デザイン：白い羽と宝石が組み合わさった、繊細なピンセットのような形。

## 📍 特殊アイテム (Special Items)

*   **ターゲットピン (Target Pin)**
    *   デザイン：非常に目立つ「赤い押しピン」。中心に星印がある。
*   **仮想探索卓 (Block)**
    *   Top: 大きな地図が広げられ、コンパスと羽ペンが置かれている。
    *   Side: 木材と金属パイプが組み合わさった、スチームパンク風のデスク。

## 🖥️ GUI (Graphical User Interface)
*   **背景**: 古い地図が敷かれた木製のテーブル。
*   **プログレスバー**: 羽ペンが地図に線を引いていくようなアニメーション。
*   **エネルギーバー**: ガラス管に入った赤いエネルギー（溶岩やレッドストーンエネルギー）。
*   **流体バー**: ガラス管に入った青い液体。
